# Screw-Jack-Design-MATLAB

## Overview
This project is a user input–based MATLAB program developed for the design and validation of a screw jack using standard machine design equations.

## Features
- User-defined input parameters
- Automatic calculation of screw jack dimensions
- Stress and safety validation
- Torque, efficiency, and self-locking analysis
- MATLAB-generated design outputs

## Parameters Considered
- Load
- Lift
- Material properties
- Factor of safety
- Permissible stresses
